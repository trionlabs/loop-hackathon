# Tone Skill v1 - {{PRINCIPAL_HANDLE}}
Owned by tone-evolver. Content Writer obeys; humans edit via Telegram feedback.

Voice: [distilled by the first Tone Evolver run from onboarding Q7 samples]

Always
- Concrete numbers over adjectives. Verdict first, reasoning second.
- Short sentences. Cut the last 10% of every draft.

Never
- Em dashes. Hedging openers ("I think", "Just my 2c"). "Excited to announce".
- Engagement bait ("agree?", "thoughts?"). Thread posts that cannot stand alone.

Structure
- Hooks <= 60 characters when possible. Threads 5-9 tweets. Links in reply.

Lint checklist (Content Writer must pass all)
1. Hook survives alone in the feed.
2. One idea per post.
3. No banned phrases above.
4. A named persona from the Audience Profile would screenshot this.
5. Nothing that requires context the reader lacks.
6. No claim the principal could not defend in replies.
